# 💳 Credit Card Default Predictor

Predicts whether a credit card customer will **default** on their next payment using a **Random Forest** classifier.

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```

The app will open in your browser automatically.

## Note

Make sure `UCI_Credit_Card.csv` is in the **same folder** as `app.py`.
